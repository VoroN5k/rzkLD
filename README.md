# 🎬 HLS Downloader

Маленька програма, яка вміє зберігати відео, що транслюється потоково (HLS / `.m3u8`), у звичайний `.mp4` файл на твоєму комп'ютері.

## 📥 Встановлення

1. Перейди на сторінку **[Releases](../../releases)**
2. Завантаж останню версію — файл `.msi` (інсталятор) або `.exe` (запускається напряму, без встановлення)
3. Якщо Windows покаже попередження "Невідомий видавець" — це нормально для невеликих застосунків без платного сертифіката. Тисни "Виконати все одно" / "More info → Run anyway"

## ⚙️ Обов'язкова умова: ffmpeg

Програма використовує **ffmpeg** для збирання відео у фінальний `.mp4`. Без нього не запрацює.

**Як встановити (Windows):**
```
winget install ffmpeg
```
Якщо `winget` не знайдено — завантаж вручну з [ffmpeg.org](https://ffmpeg.org/download.html) і додай шлях до `ffmpeg.exe` у системну змінну PATH.

Перевірити, що встановилось:
```
ffmpeg -version
```

Якщо команда покаже версію — все ок.

## ▶️ Як користуватись

1. Відкрий програму
2. Встав посилання на `.m3u8`-плейлист у поле **URL**
3. (опціонально) зміни назву вихідного файлу
4. Тисни **"Завантажити"**
5. Зачекай, поки прогрес-бар дійде до 100% і програма сконвертує файл у mp4
6. Готове відео з'явиться в тій же папці, де лежить програма

## 🤖 Автопошук плейлиста (для власного плеєра)

Якщо в тебе свій плеєр (наприклад React-застосунок із hls.js), і ти не хочеш
шукати `.m3u8` вручну через DevTools — встав URL сторінки плеєра в поле
**"URL сторінки плеєра"**, опціонально вкажи CSS-селектор кнопки play (якщо
відео не стартує само), і тисни **"🔍 Знайти m3u8"**. Програма відкриє
сторінку в headless-браузері, натисне кнопку і автоматично підставить
знайдене посилання в поле для завантаження.

**Вимога:** на комп'ютері має бути встановлений справжній Google Chrome або
Chromium (звичайний, не з Microsoft Store — той не підтримує потрібний
протокол automation).

## 🔍 Де взяти посилання на `.m3u8`

1. Відкрий відео в браузері
2. Натисни `F12` (DevTools) → вкладка **Network**
3. Введи у фільтр `m3u8`
4. Запусти відтворення відео
5. У списку з'явиться запит, що повертає текст, який починається з `#EXTM3U` — це і є потрібне посилання

## ⚠️ Важливо

Програма — це звичайний технічний інструмент для роботи з HLS-потоками (як `ffmpeg` чи `yt-dlp`). Використовуй її тільки для контенту, на перегляд/збереження якого маєш право: власні відео, відкриті/публічні джерела, легальні сервіси, що дозволяють офлайн-перегляд.

## 🐛 Якщо щось не працює

- **"ffmpeg не знайдено в PATH"** — встанови ffmpeg (див. вище) і перезапусти програму
- **Завантаження зависає на 0%** — перевір, що посилання справді веде на `.m3u8`-файл (відкрий його прямо в браузері, має показати текст, що починається з `#EXTM3U`)
- **Файл завантажився, але не відкривається** — спробуй відкрити через VLC; якщо й там не йде, можливо потік зашифрований (AES-128) — ця версія програми такого не підтримує
